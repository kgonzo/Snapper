Guide to downloading and running the snapper application
	1. Open eclipse
	2. Under file, click open projects from file system
	3. Choose the archive option (next to directory option)
	4. Find the snapper zip file, and select that
	5. Finish the import (it's ok if it is imported as Snapper.zip expanded)
	6. Within Snapper in the package explorer, navigate under Snapper.src
	7. Navigate under the application package
	8. Run the Main.java file
